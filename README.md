# 使用方式

1. 下载插件
   - [Releases](https://github.com/cofedream/extensions-copymanga/releases)下载插件
   - 浏览-右上角-插件仓库-添加 输入以下网址`https://raw.githubusercontent.com/cofedream/extensions-copymanga/repo/index.min.json`
2. 打开Mihon-更多-设置-高级
   - 清除Cookie
   - 清除Webview缓存
   - 默认UA设置
